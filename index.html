How It Works

DelTriC uses the mathematical structure of Delaunay triangulation to find clusters — connecting nearby points and cutting edges that bridge different clusters or cross outlier regions.

1

Project to 2D

High-dimensional data is projected to 2D using UMAP (preserves local structure) or PCA (fast, linear). This makes Delaunay triangulation tractable.

2

Delaunay Triangulation

Every point is connected to its natural neighbors, forming a mesh of triangles. The result is a complete graph where cluster structure is encoded in edge lengths.

3

Neighbor-Bridge Pruning

Edges are pruned using local statistics: for each point, we compute the median and IQR of its neighbor edge lengths. Edges longer than median + prune_param x IQR are cut. This makes the algorithm naturally robust to varying densities.

4

Connected Components → Clusters

The remaining graph's connected components are the clusters. Small components (≤ 5 points) are labeled as outliers (−1).

5

Outlier Refinement

A final pass merges borderline anomaly points into nearby clusters or keeps them as outliers based on the anomaly_sensitivity parameter.

Install & Use

Install from PyPI and start clustering in 3 lines.

$ pip install deltric

Quick Start

from deltric import DelTriC
import numpy as np

X = np.random.randn(500, 10)

# Auto-param mode — just pass your data
model = DelTriC()
labels = model.fit_predict(X)

n_clusters = len(set(labels) - {-1})
n_outliers = np.sum(labels == -1)
print(f"Found {n_clusters} clusters, {n_outliers} outliers")

Advanced Usage

# Force PCA projection, tune pruning
model = DelTriC(
    dim_reduction="pca",
    prune_param=1.3,
    anomaly_sensitivity=0.3,
    min_cluster_size=10,
    random_state=42,
)
labels = model.fit_predict(X)
print(model._last_auto_params())

In a scikit-learn Pipeline

from sklearn.pipeline import Pipeline
from sklearn.preprocessing import StandardScaler

pipe = Pipeline([
    ("scale", StandardScaler()),
    ("cluster", DelTriC(dim_reduction="pca")),
])
labels = pipe.fit_predict(X)
