How it works

DelTriC builds a Delaunay triangulation over the data, then uses a novel neighbor-bridge pruning strategy to separate clusters from outliers — all without assuming uniform density.

1

Project to lower dimension

Optionally reduce dimensionality with PCA or UMAP to make the Delaunay triangulation tractable and robust.

2

Build Delaunay triangulation

Construct the triangulation over the projected points. Each edge connects natural neighbors in the geometric sense.

3

Neighbor-bridge pruning

Edges are classified as within-cluster, between-cluster, or outlier edges using local density statistics. Outlier edges are pruned.

4

Extract clusters & outliers

Connected components of the remaining graph form clusters. Points connected only by pruned edges are flagged as outliers.

Install & Use

Quick Start

from deltric import DelTriC
import numpy as np

# X: numpy array of shape (n_samples, n_features)
X = np.random.randn(500, 10)

# Auto-param mode — just pass your data
model = DelTriC()
labels = model.fit_predict(X)

n_clusters = len(set(labels) - {-1})
n_outliers = np.sum(labels == -1)
print(f"Found {n_clusters} clusters, {n_outliers} outliers")

Advanced Usage

# Force PCA projection, tune pruning
model = DelTriC(
    dim_reduction="pca",
    prune_param=1.3,
    anomaly_sensitivity=0.3,
    min_cluster_size=10,
    random_state=42,
)
labels = model.fit_predict(X)
print(model._last_auto_params())

In a scikit-learn Pipeline

from sklearn.pipeline import Pipeline
from sklearn.preprocessing import StandardScaler

pipe = Pipeline([
    ("scale", StandardScaler()),
    ("cluster", DelTriC(dim_reduction="pca")),
])
labels = pipe.fit_predict(X)
